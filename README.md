## Project Overview
This repository contains the complete database implementation for the
SKS National Bank case study.

The project includes:
- Database design (ERD)
- Database creation and population scripts
- Prepared stored procedures and functions
- Advanced features (users, privileges, triggers, audit logging)
- JSON and spatial data support
- Final database backup
